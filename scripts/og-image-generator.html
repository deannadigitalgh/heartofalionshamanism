🦁 How to Generate Your Open Graph Image

  1. Capture the canvas area above: Use your browser's screenshot tool or an extension like "Full Page Screen Capture" (Chrome) or "Lightshot".
  2. Screenshot settings: Make sure to capture only the colored gradient box, not the instructions below it. The visible content should be approximately 920x546 pixels in your browser view.
  3. Crop and resize: Crop tightly around the canvas area (the gradient background with text). Then use an image editor to resize exactly to 1200 x 630 pixels. This is the standard Open Graph size for Facebook, Twitter, LinkedIn.
  4. Save as JPG: Export/save your final image as a JPG file with approximately 85% quality.
  5. Name and place it: Rename to heart-of-lion-og.jpg and move it into the project's /public/ folder.
Or use Canva (recommended for easier editing)
