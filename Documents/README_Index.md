## How to Use This Index

### For Website Development:
1. **Homepage Content:** Start with 01_Client_Profile.md for overview, then pull specific details from other files as needed
2. **About Pages:** Use 01_Client_Profile.md + 02_Shamanic_Practice.md (shamanism site) or 03_Acupuncture_Practice.md (acupuncture site)
3. **Services Pages:** Reference 02_Shamanic_Practice.md and 03_Acupuncture_Practice.md for service descriptions
4. **Workshops Pages:** Use 05_Workshops_Events.md for complete workshop information
5. **Testimonials/FAQ:** Combine information from all files as needed

### For SEO Strategy:
- Extract location-specific keywords from practice addresses and service areas
- Use credential information from all files for authority building
- Reference book title and workshops for content marketing

### For Client Communication:
- Keep 01_Client_Profile.md handy for quick reference to contact info and workflow
- Use other files when discussing specific services or answering client questions
